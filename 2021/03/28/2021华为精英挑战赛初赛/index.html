2021华为精英挑战赛初赛

本文最后更新于：7 分钟前

2021华为精英挑战赛初赛题目,思想以及代码总结（仅仅为baseline）

初赛题目解析

秉着化繁为简的想法，简单的阐述题目要求:

服务器

我们具有多台可供购买的服务器，每台服务器具有自己的CPU核数，内存大小，硬件成本和每天能耗成本，同时每台服务器具有双节点A和B，每个节点分担服务器资源的一半:

以NV603为例，其CPU核数，内存大小，硬件成本和每天能耗成本分别为92C，324G，53800，500。其中，硬件成本只是在购买该台服务器时候需要花费，而每天能耗成本则在服务器运行当天存在，换言之，若服务器当天没有被部署，则不需要计算每天能耗成本。每台服务器具有两个节点，每个节点分别具有该台服务器一半的资源：A具有CPU核数46C，内存大小162G.

虚拟机

此外，我们还提供了可供购买的虚拟机数台，每台虚拟机具有所需要的CPU核数，内存大小和是否双节点部署:

服务器所剩余的CPU核数核内存大小足够的情况下，该台虚拟机才能部署进该服务器。如果该台虚拟机是单节点部署，那么该台虚拟机将部署在服务器的A或者B节点，也将占据A或者B节点的资源。反之，如果虚拟机是双节点部署，那么该台虚拟机将占据服务器A和B节点的一半资源。例如C3.8xlarge.2, 所部署的服务器的A和B节点将分别提供一半的资源，亦即，A和B节点分别提供16C和32G资源。

资源调度和分配

  1. 容量约束： 服务器可以用来容纳用户的虚拟机，但是服务器上的任意一个节点(A 和 B)上的资源负载(CPU 和内存)均不能超过其容量上限。
  2. 请求类型： 用户的请求共分为两种类型：创建请求和删除请求。创建请求表示用户向公有云平台提交的一个创建虚拟机的请求；删除请求表示用户提交的删除一台之前创建的虚拟机的请求。
  3. 请求序列： 由一系列请求构成的序列。题目会给出接下来若干天中每一天用户的请求序列，根据每天的请求序列，你需要进行相应的资源规划和调度。
  4. 数据中心扩容： 在得知了一天的请求序列后，你可以在实际进行调度前进行一次数据中心扩容。即购买一些新的服务器来容纳后续用户请求的虚拟机，同时你需要付出所购买服务器相应的硬件成本。你需要指定购买哪些类型的服务器以及购买的数量。初始时你没有任何服务器。
  5. 虚拟机迁移： 在完成扩容后，在处理每一天的新请求之前，你还可以对当前存量虚拟机进行一次迁移，即把虚拟机从一台服务器迁移至另一台服务器。对于单节点部署的虚拟机，将其从一台服务器的 A 节点迁移至 B 节点(或反之)也是允许的。迁移的目的服务器和节点必须有足够的资源容纳所迁移的虚拟机。迁移的虚拟机总量不超过当前存量虚拟机数量的千分之五。即假设当前有 n 台存量虚拟
    机，每天你可以迁移的虚拟机总量不得超过 5n/1000 向下取整。
  6. 部署虚拟机： 在完成扩容和迁移之后，你需要按顺序处理当天所有的新请求。对于每一个创建虚拟机的新请求，你要为虚拟机指定一台服务器进行部署。若虚拟机是单节点部署的，你还需要指明部署在服务器的 A 节点还是 B 节点。处理请求的过程中，任意一台服务器上每个节点容纳的虚拟机资源总和都不能超出节点本身的资源容量（指 CPU 和内存两个维度）。

输入示例：

  • 第一行代表输入的可供购买的服务器的行数N
  • 接近着的N行代表N种可供购买的服务器类型及其规格（服务器型号，CPU核数，内存大小，硬件成本，每日成本）
  • 再输入的代表可供部署的虚拟机类型M
  • 接近着的N行代表M种可供购买的虚拟机类型及其规格（服务器型号，CPU核数，内存大小，是否双节点部署）
  • 存在请求的天数
  • 第一天的请求量（K）
  • K条请求
  • 第二天的请求量（L）
  • L条请求
  • …..

输出示例：

  • 首先第一行输出(purchase, Q)，其中 Q 是一个整数表示你需要扩容购买多少种类型的服务器。 Q 不能为负数且不能大于 N。
  • 接下来 Q 行，每行的格式为： (服务器型号, 购买数量)。 例如(NV603, 2)表示扩容购买两台 NV603 服务器。购买数量必须为正整数，在整个用例中，你所购买的服务器总量不能超过 105。
  • 你所购买的每一台服务器都会被分配一个编号，从零开始。例如你在第一天输出了
    (NV603, 2)
    (NV604, 1)
    则你会拥有两台 NV603 服务器，编号分别为 0 和 1；一台 NV604 服务器，编号为2。在第二天输出了
    (NV603, 3)
    则会新增三台 NV603 服务器，编号分别为 3， 4， 5。 你所购买的每一台服务器都会被分配一个编号，从零开始。例如你在第一天输出了
    (NV603, 2)
    (NV604, 1)
    则你会拥有两台 NV603 服务器，编号分别为 0 和 1；一台 NV604 服务器，编号为2。在第二天输出了
    (NV603, 3)
    则会新增三台 NV603 服务器，编号分别为 3， 4， 5。
  • 接下来一行输出(migration, W)，其中 W 是一个整数表示你要迁移的虚拟机的数量。
  • 接下来 W 行，每行表示一个虚拟机的迁移，格式为(虚拟机 ID, 目的服务器 ID)或(虚拟机 ID, 目的服务器 ID, 目的服务器节点)。例如(3, 1)表示将 ID 为 3 的虚拟机从当前所在服务器迁移至 ID 为 1 的服务器，该虚拟机必须是双节点部署的； (4, 1,A)表示将 ID 为 4 的虚拟机从当前所在服务器迁移至 ID 为 1 的服务器的 A 节点，该虚拟机必须是单节点部署的。
  • 接下来按输入中请求的顺序，输出对于当前这一天的每一个创建请求，该虚拟机部署的服务器 ID，格式为(服务器 ID)或者(服务器 ID, 部署节点)。对于双节点部 署的虚拟机，你只需要输出其部署的服务器 ID，对于单节点部署的虚拟机，你还需要输出部署的节点(A 或 B)。

实现思想

本次比赛的baseline主要采用贪心的思想，尽可能选取能满足当天需求的性价比高的服务器。因此，需要对可供购买的服务器按照性价比进行排序。

//对可供购买的服务器链表按照性价比降序冒泡排序
void orderServer(Sever severs[],int serversLen){
    Sever temp;
    for(int i=0;i<serversLen-1;i++){
        for(int j=i+1;j<serversLen;j++){
            double valueRatioI = (double)(severs[i].cpuCodes+severs[i].memorySize)/(double)(severs[i].powerCost+(double)severs[i].cost/1000);
            double valueRatioJ = (double)(severs[j].cpuCodes+severs[j].memorySize)/(double)(severs[j].powerCost+(double)severs[j].cost/1000);
            if(valueRatioI<valueRatioJ){
                temp = severs[i];
                severs[i] = severs[j];
                severs[j] = temp;
            }
        }
    }
}

//处理add请求的同时判断是否需要购买
bool isNeedPurchase(list<PurchasedSevers>* purchasedSevers,VM vm){
    //判断该条添加虚拟机请求服务器链表是否能放得下，不能就购买一次
    for(list<PurchasedSevers>::iterator iter=(*purchasedSevers).begin();iter!=(*purchasedSevers).end();iter++){
        if(!vm.isDoubleNode){
            if(vm.memorySize<=iter->ARemainMemorySize&&vm.cpuCodes<=iter->ARemainCores){
                iter->ARemainMemorySize-=vm.memorySize;
                iter->ARemainCores-=vm.cpuCodes;
                vm.aorb = 0;
                iter->vmSet.push_back(vm);
                string s1 = "("+int2str(iter->ID),s2 = ", A)";
                result.push_back(s1+s2);
                return false;
            }
            if(vm.memorySize<=iter->BRemainMemorySize&&vm.cpuCodes<=iter->BRemainCores){
                iter->BRemainMemorySize-=vm.memorySize;
                iter->BRemainCores-=vm.cpuCodes;
                vm.aorb = 1;
                iter->vmSet.push_back(vm);
                string s1 = "("+int2str(iter->ID),s2 = ", B)";
                result.push_back(s1+s2);
                return false;
            }
        }
        else{
            if(vm.memorySize/2<=iter->ARemainMemorySize&&vm.cpuCodes/2<=iter->ARemainCores&&vm.memorySize/2<=iter->BRemainMemorySize&&vm.cpuCodes/2<=iter->BRemainCores){
                iter->ARemainMemorySize-=vm.memorySize/2;
                iter->ARemainCores-=vm.cpuCodes/2;
                iter->BRemainMemorySize-=vm.memorySize/2;
                iter->BRemainCores-=vm.cpuCodes/2;
                vm.aorb = 2;
                iter->vmSet.push_back(vm);
                string s1 = "("+int2str(iter->ID)+")";
                result.push_back(s1);
                return false;
            }
        }
    }
    return true;
}


//购买满足当天请求的服务器
void purchaseSever(Sever severs[],int seversLen,string require[],VM vms[],list<PurchasedSevers>* purchasedSevers) {
    if(require[0]=="")return;
    Sever needPurchase[SEVERSNUM];
    int needPurchaseNum[SEVERSNUM];
    int needPurchaseLen = 0,index,di;
    for (int i= 0; i < MAXREQUIREPERDAY; i++) {
        if (require[i] == "")break;
        vector<string> tempS = split(require[i], ',');
        if (tempS[0][0] == 'a') {
            string tempSSub = tempS[1].substr(3, tempS[1].size() - 2);
            di=0;
            index = getHashCodeIndex(tempSSub, vmsNum, 0);
            while (vms[index].type != tempSSub) {
                index = getHashCodeIndex(tempSSub, vmsNum, ++di);
            }
            if (tempS[0][0] == 'a') {
                //找到需要添加的虚拟机并设置ID
                vms[index].ID = atoi(tempS[2].c_str());
                VM currentVm = vms[index];
                //判断是否需要购买服务器
                if (isNeedPurchase(purchasedSevers, currentVm)){
                    //开始购买
                    for (int j = 0; j < seversLen; j++) {
                        bool coreFlag, memoryFlag;
                        if (!currentVm.isDoubleNode) {
                            //如果是单节点虚拟机
                            coreFlag = severs[j].cpuCodes/2 >= currentVm.cpuCodes;
                            memoryFlag = severs[j].memorySize/2 >= currentVm.memorySize;
                        } else {
                            //如果是双节点虚拟机
                            coreFlag = severs[j].cpuCodes/2 >= currentVm.cpuCodes / 2;
                            memoryFlag = severs[j].memorySize/2 >= currentVm.memorySize / 2;
                        }
                        //当前服务器类型满足该虚拟机的核数和内存需求
                        if (coreFlag && memoryFlag) {
                            //需要购买的虚拟机映射表中已经有该虚拟机，则购买数量加一
                            int k;
                            for (k = 0; k < needPurchaseLen; k++) {
                                if (needPurchase[k].type == severs[j].type) {
                                    needPurchaseNum[k] += 1;
                                    break;
                                }
                            }
                            //需要购买的虚拟机映射表中没有该虚拟机，加入该虚拟机并设置购买数量为初始1
                            if(k>=needPurchaseLen){
                                needPurchase[needPurchaseLen] = severs[j];
                                needPurchaseNum[needPurchaseLen] = 1;
                                needPurchaseLen++;
                            }
                            //这部分是将需要购买的服务器添加至已购买服务器，同时维护服务器递增链表
                            //新服务器添加进已购买服务器数组
                            PurchasedSevers temp;
                            setPurchasedSevers(&severs[j], &temp);
                            (*purchasedSevers).push_back(temp);
                            break;
                        }
                    }
                    isNeedPurchase(purchasedSevers, currentVm);
                }
            }
        }
        else{
            //如果是删除操作
            int delVMID = atoi(tempS[1].c_str());
            bool removeFlag=false;
            for(list<PurchasedSevers>::iterator iter=(*purchasedSevers).begin();iter!=(*purchasedSevers).end();iter++){
                for(list<VM>::iterator vmIter=iter->vmSet.begin();vmIter!=iter->vmSet.end();vmIter++){
                    if(vmIter->ID==delVMID){
                        di = 0;
                        index=getHashCodeIndex(vmIter->type,vmsNum,0);
                        while (vms[index].type != vmIter->type) {
                            index = getHashCodeIndex(vmIter->type,vmsNum, ++di);
                        }
                        if(vmIter->aorb==0){
                            iter->ARemainCores+=vmIter->cpuCodes;
                            iter->ARemainMemorySize+=vmIter->memorySize;
                        }
                        else if(vmIter->aorb==1){
                            iter->BRemainCores+=vmIter->cpuCodes;
                            iter->BRemainMemorySize+=vmIter->memorySize;
                        }
                        else if(vmIter->aorb==2){
                            iter->ARemainCores+=(vmIter->cpuCodes/2);
                            iter->ARemainMemorySize+=(vmIter->memorySize/2);
                            iter->BRemainCores+=(vmIter->cpuCodes/2);
                            iter->BRemainMemorySize+=(vmIter->memorySize/2);
                        }
                        iter->vmSet.erase(vmIter);
                        removeFlag=true;
                    }
                    if(removeFlag)break;
                }
                if(removeFlag)break;
            }
        }
    }

    //双For循环遍历needPurchase的类型及其个数，更新_ID

    //打印输出_ID;
    //打印输出
    cout << "(purchase, " << needPurchaseLen << ")" << endl;
    for (int i = 0; i < needPurchaseLen; i++) {
        cout << "(host" << needPurchase[i].type << ", " << needPurchaseNum[i] << ")" << endl;
    }
    moveVM(purchasedSevers);
    for (list<string>::iterator iter=result.begin();iter!=result.end();iter++) {
        cout << iter->data() << endl;
    }
    result.clear();
    /*for (list<string>::iterator iter1=result.begin();iter1!=result.end();iter1++) {
        result.erase(iter1);
    }*/
}C

vector<string> split(string str,char del)
{
    stringstream ss(str);
    string tok;
    vector<string> ret;
    while(getline(ss,tok,del))
    {
        if(tok > "")
            ret.push_back(tok);
    }
    return ret;
};

//生成hash值，s是字符串，serorvm是服务器或虚拟机对应的结构体数组大小，di是偏移量
int getHashCodeIndex(string s,int serorvm, int di){
    int res=0;
    for(int i=0;i<s.length();i++){
        res+=s[i];
    }
    return (res+di)%serorvm;
}

            if(vm.memorySize/2<=iter->ARemainMemorySize&&vm.cpuCodes/2<=iter->ARemainCores&&vm.memorySize/2<=iter->BRemainMemorySize&&vm.cpuCodes/2<=iter->BRemainCores){
                iter->ARemainMemorySize-=vm.memorySize/2;
                iter->ARemainCores-=vm.cpuCodes/2;
                iter->BRemainMemorySize-=vm.memorySize/2;
                iter->BRemainCores-=vm.cpuCodes/2;
                vm.aorb = 2;
                iter->vmSet.push_back(vm);
                string s1 = "("+int2str(iter->ID)+")";
                result.push_back(s1);
                return false;
            }
        }
    }
    return true;
}

//对可供购买的服务器链表按照性价比降序冒泡排序
void orderServer(Sever severs[],int serversLen){
    Sever temp;
    for(int i=0;i<serversLen-1;i++){
        for(int j=i+1;j<serversLen;j++){
            double valueRatioI = (double)(severs[i].cpuCodes+severs[i].memorySize)/(double)(severs[i].powerCost+(double)severs[i].cost/1000);
            double valueRatioJ = (double)(severs[j].cpuCodes+severs[j].memorySize)/(double)(severs[j].powerCost+(double)severs[j].cost/1000);
            if(valueRatioI<valueRatioJ){
                temp = severs[i];
                severs[i] = severs[j];
                severs[j] = temp;
            }
        }
    }
}

void purchaseSever(Sever severs[],int seversLen,string require[],VM vms[],list<PurchasedSevers>* purchasedSevers) {
    if(require[0]=="")return;
    Sever needPurchase[SEVERSNUM];
    list<PurchasedSevers> purchaseSeversPerDay;
    int needPurchaseNum[SEVERSNUM];
    int needPurchaseLen = 0,index,di;
    for (int i= 0; i < MAXREQUIREPERDAY; i++) {
        if (require[i] == "")break;
        vector<string> tempS = split(require[i], ',');
        if (tempS[0][0] == 'a') {
            string tempSSub = tempS[1].substr(3, tempS[1].size() - 2);
            di=0;
            index = getHashCodeIndex(tempSSub, vmsNum, 0);
            while (vms[index].type != tempSSub) {
                index = getHashCodeIndex(tempSSub, vmsNum, ++di);
            }
            if (tempS[0][0] == 'a') {
                //找到需要添加的虚拟机并设置ID
                vms[index].ID = atoi(tempS[2].c_str());
                VM currentVm = vms[index];
                //判断是否需要购买服务器
                if (isNeedPurchase(purchasedSevers, currentVm)){
                    //开始购买
                    for (int j = 0; j < seversLen; j++) {
                        bool coreFlag, memoryFlag;
                        if (!currentVm.isDoubleNode) {
                            //如果是单节点虚拟机
                            coreFlag = severs[j].cpuCodes/2 >= currentVm.cpuCodes;
                            memoryFlag = severs[j].memorySize/2 >= currentVm.memorySize;
                        } else {
                            //如果是双节点虚拟机
                            coreFlag = severs[j].cpuCodes/2 >= currentVm.cpuCodes / 2;
                            memoryFlag = severs[j].memorySize/2 >= currentVm.memorySize / 2;
                        }
                        //当前服务器类型满足该虚拟机的核数和内存需求
                        if (coreFlag && memoryFlag) {
                            //需要购买的虚拟机映射表中已经有该虚拟机，则购买数量加一
                            int k;
                            for (k = 0; k < needPurchaseLen; k++) {
                                if (needPurchase[k].type == severs[j].type) {
                                    needPurchaseNum[k] += 1;
                                    break;
                                }
                            }
                            //需要购买的虚拟机映射表中没有该虚拟机，加入该虚拟机并设置购买数量为初始1
                            if(k>=needPurchaseLen){
                                needPurchase[needPurchaseLen] = severs[j];
                                needPurchaseNum[needPurchaseLen] = 1;
                                needPurchaseLen++;
                            }
                            //这部分是将需要购买的服务器添加至已购买服务器，同时维护服务器递增链表
                            //新服务器添加进已购买服务器数组
                            PurchasedSevers temp;
                            setPurchasedSevers(&severs[j], &temp);
                            (*purchasedSevers).push_back(temp);
                            purchaseSeversPerDay.push_back(temp);
                            break;
                        }
                    }
                    isNeedPurchase(purchasedSevers, currentVm);
                }
            }
        }
        else{
            //如果是删除操作
            int delVMID = atoi(tempS[1].c_str());
            bool removeFlag=false;
            for(list<PurchasedSevers>::iterator iter=(*purchasedSevers).begin();iter!=(*purchasedSevers).end();iter++){
                for(list<VM>::iterator vmIter=iter->vmSet.begin();vmIter!=iter->vmSet.end();vmIter++){
                    if(vmIter->ID==delVMID){
                        di = 0;
                        index=getHashCodeIndex(vmIter->type,vmsNum,0);
                        while (vms[index].type != vmIter->type) {
                            index = getHashCodeIndex(vmIter->type,vmsNum, ++di);
                        }
                        if(vmIter->aorb==0){
                            iter->ARemainCores+=vmIter->cpuCodes;
                            iter->ARemainMemorySize+=vmIter->memorySize;
                        }
                        else if(vmIter->aorb==1){
                            iter->BRemainCores+=vmIter->cpuCodes;
                            iter->BRemainMemorySize+=vmIter->memorySize;
                        }
                        else if(vmIter->aorb==2){
                            iter->ARemainCores+=(vmIter->cpuCodes/2);
                            iter->ARemainMemorySize+=(vmIter->memorySize/2);
                            iter->BRemainCores+=(vmIter->cpuCodes/2);
                            iter->BRemainMemorySize+=(vmIter->memorySize/2);
                        }
                        iter->vmSet.erase(vmIter);
                        removeFlag=true;
                    }
                    if(removeFlag)break;
                }
                if(removeFlag)break;
            }
        }
    }

    //双For循环遍历needPurchase的类型及其个数，更新_ID

    static int _IDIncre=0;
    static unordered_map<int,int> ID_Map;
    //bool increIter = true;
    for(list<PurchasedSevers>::iterator iter1 = purchaseSeversPerDay.begin();iter1!=purchaseSeversPerDay.end();iter1++){
        //if(prePurchaseNum!=0)advance(iter1,prePurchaseNum-1);
        /*if(increIter){
            for(int i=0;i<curSize;i++){
                iter1++;
            }
            increIter=false;
        }*/
        for(list<PurchasedSevers>::iterator iter2 = iter1;iter2!=purchaseSeversPerDay.end();iter2++){
            //if(prePurchaseNum!=0)advance(iter2,prePurchaseNum-1);
            if(iter2->type==iter1->type && ID_Map.count(iter2->ID)==0){
                ID_Map.insert(make_pair(iter2->ID,_IDIncre++));
            }
        }
    }
    //打印输出_ID;
    //打印输出
    cout << "(purchase, " << needPurchaseLen << ")" << endl;
    for (int i = 0; i < needPurchaseLen; i++) {
        cout << "(host" << needPurchase[i].type << ", " << needPurchaseNum[i] << ")" << endl;
    }
    moveVM(purchasedSevers);
    for (list<string>::iterator iter=result.begin();iter!=result.end();iter++) {
        string s = iter->data();
        if(s.find(',')!=string::npos){
            int tempID=atoi(s.substr(1,s.size()-5).c_str());
            s.replace(1,s.size()-5,int2str(ID_Map.find(tempID)->second));

            int tempID=atoi(s.substr(1,s.size()-2).c_str());

        }
        cout<< s << endl;

    }*/
